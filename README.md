# Elevate

Elevate is a web application that allows you to track hikes. You can add future hikes to a wishlist and check them off as you go. This application allows users to create an account to store their hikes as well. The application also also tracks and displays stats such as # of hikes completed and total miles hiked.

## Instructions for Build and Use

Steps to build and/or run the software:

1. Install Live server VS Code extension
2. Install firebase
2. Run index.html with Live server
3. Application will open in a browser

Instructions for using the software:

1. Create an account using the button in the top left corner
2. Add trails using the "add trails" button in the navigation bar
3. Mark trails comleted in the "view trails" page
4. View your stats in the left column and your wishlist in the right column

## Development Environment

To recreate the development environment, you need the following software and/or libraries with the specified versions:

* Live Server
